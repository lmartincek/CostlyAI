# CostlyAI - Backend API
This is the backend for [CostlyAI](https://www.github.com/lmartincek/CostlyAI-webclient). It provides a RESTful API for interacting with the database and OpenAI API.

## Technologies used

- Express.js
- TypeScript
- Supabase
- OpenAI API
